# **App Name**: MCOBUILDLAB

## Core Features:

- Wizard-Based Server Configuration: A multi-step selection interface for server name, type, size, style, and security levels using specialized form inputs.
- AI Architecture Tool: Generates a complete server analysis, including category and channel structures, using an AI tool based on the user's selected preferences.
- Role & Hierarchy Visualizer: Displays the generated role list with their corresponding HEX colors and hierarchy in a structured, clean list format.
- Bot Recommendation Engine: An AI-driven engine that matches the best Discord bots to the selected server features and type automatically.
- Interactive Permissions Map: View and audit the security levels and channel permissions for every category within the proposed structure.
- Configuration JSON Exporter: Exports the full server specification as a production-ready JSON object compatible with Discord templating tools.

## Style Guidelines:

- Primary color: Architectural Cobalt (#3d8bff) representing professional planning and depth.
- Background: Deep Midnight Navy (#14171c) for a sleek, dark mode aesthetic focused on the interface content.
- Accent: Electric Blueprint Cyan (#1ad1ff) for highlight indicators, buttons, and call-to-action elements.
- Headline font: 'Space Grotesk' for a computerized, technical feel suitable for architectural blueprints. Body font: 'Inter' for high legibility and an objective, machined appearance.
- Outline icons with a high-stroke weight for a geometric and structured look, matching the professional architect vibe.
- Step-based modal transitions with progress tracking, utilizing a clean grid system for the analysis report results.
- Smooth slide-in transitions between form steps and an elegant skeleton-loading state while the AI generates the server structure.
